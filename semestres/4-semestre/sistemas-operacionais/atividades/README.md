# Atividades

Espaco para atividades, listas e exercicios da UC de Sistemas Operacionais.

## Estrutura atual

- atividade-iii/
- atividade-iv/
- gerar_pdfs_entrega.py

Cada pasta de atividade segue o mesmo padrao:

- enunciado/
	- atividade-xxx-enunciado.pdf
- entrega/
	- atividade-xxx-entrega-final.pdf

Esse formato evita nomes ambíguos e facilita a validacao automatica.

O script na raiz valida se cada pasta contem os PDFs esperados, inclusive em subpastas.
